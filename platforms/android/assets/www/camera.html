Camera API

Here we have a sample of a photo camera app, using an API from Apache Cordova framework in order to activate device’s native functionality. The user may access it’s own library and album folders. The Caputure Edit button makes a smaller photo that could be used to edit in another application while the Capture button will get a photo with better quality but higher size. I intend to develop a Slider where he photos captured could be displayed and posted using social network or at the official Zetto website.
